Part 4/extension: Build/code to make a difference

Environment bot - Seed hole digger - Modify the code for the same build.

  • Can you modify your code to make your robot dig multiple holes, with a few seconds pause between each hole, to move to a new spot? (scattered blocks)?

Health care bot - Hospital cleaner - Modify the build for the same code.

  • In order to sweep and mop around the hospital. Can you modify the build to make effective tools for this job?

Agriculture bot - Hole digger - Modify the code and the build.

  • Your robot is now a fence post digger. Can you modify your build’s tool to dig holes of different sizes (small block piles)?
